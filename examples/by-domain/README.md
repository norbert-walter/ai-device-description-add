# Examples by Domain

ADD document examples organized by application domain. Each example shows how ADD is applied in a specific real-world context — from home automation to critical infrastructure.

| Domain | Examples |
|---|---|
| [home-automation](home-automation/) | Garden irrigation valve, climate sensor |
| [workshop-industry](workshop-industry/) | CNC milling machine, dust extraction, office access control |
| [medical](medical/) | Hospital bed vital signs monitor |
| [infrastructure](infrastructure/) | Medium voltage distribution switch |
| [critical](critical/) | Nuclear primary coolant pump (read-only monitoring) |

All examples are self-contained and validated. The domain determines the Autonomy Level — home automation typically Level 1–2, critical infrastructure Level 3.
